Installation - important

(1)Edit the libraries/joomla/error/error.php file (with help of text editor), add into the "raiseError" method (row + - 169) the following code:

//PHOCAEDIT
if ($code = '404') {
include_once( JPATH_ADMINISTRATOR.DS.'components'.DS.'com_phocasef'.DS.'helpers'.DS.'phocasef.php' );
PhocaSefHelper::createUrlEntry();
}
//PHOCAEDIT

You should paste it before:
$reference = & JError::raise(E_ERROR, $code, $msg, $info, true);

(2) Search Engine Friendly URLs needs to be enabled in Global Configuration of Joomla!

Documentation - Parameters

Enable Redirecting of sites

Links which ends with 404 Error are automatically stored in the database (if it is set in parameters). If you click on the link, you can add new link (where the site should be redirect). For successfully redirecting of the site the link must be published. Add the links in following format:

http://www.phoca.cz/some-article/ ==>some-article

http://www.phoca.cz/index.php/some-article/ ==>index.php/some-article

It means, without http:// and without the right slash (/)

Store Sites which ends with 404 Error

If yes, links which ends with 404 Error will be stored into the database. If no, links will be no more stored but the redirecting of previously added links will work.

Store Referring Sites

If yes, referring sites of links which ends with 404 Error will be stored.
